# 电脑

## 工具

#### IntelliJ IDEA

开发代码使用的 IDE，可很高效的提升代码编写速度。

[https://www.jetbrains.com/idea/](https://www.jetbrains.com/idea/)

#### 印象笔记

一款很高效的笔记软件，可以直接与手机数据同步，支持保存网页数据、搜索文件中内容、播放 PPT 、扫描书本/名片等。

[https://www.yinxiang.com](https://www.yinxiang.com)
```
吐槽：
1.手机版本不能编辑Markdown格式的笔记，跟客服沟通上线时间未知
2.Markdown格式的，只支持复制图片，其它文件格式均不支持
```

####  Jing

TechSmith出品的截图工具，支持Mac/Windows，常用的截图功能都有，免费版本。

[https://www.techsmith.com/jing-tool.html](https://www.techsmith.com/jing-tool.html)

如果使用英文操作系统的话，登录印象笔记国内版本时，需要先将操作的语言切换回`中文`，才会显示国内印象笔记的登录入口。

#### iterm2

替换 Mac 中 Terminal 的控制台工具，可以支持安装丰富的插件，来提高工作效果。

安装 git 插件，可使用较简短的完成 git 命令。如：ga 的功能与 git add 一致。

[https://www.iterm2.com/](https://www.iterm2.com/)

#### XMind Zen

XMind公司主推的思维导图工具，相比XMind，XMind Zen有更加优美的UI显示，可免费的使用的思维导图软件，支持直接导出成图片、Word 等格式。
```
官方对比解释：
Spending 3 years, we rewrote every single byte completely. XMind: ZEN is 100% compatible with XMind 8, and supports all map structures you are familiar with.
```

[https://www.xmind.net/zen/](https://www.xmind.net/zen/)

#### 1Password

密码管理工具，支持手机、PC，收费软件，可保存网站密码、License、Secure Note等。

[https://1password.com](https://1password.com)

#### brew

Mac上包管理，让你再也不用查找安装包地址和繁琐的安装步骤所费心。一键完成软件的下载、安全。

[https://brew.sh/](https://brew.sh/)

#### Chrome 浏览器

谁用谁知道，高效、方便进行网页开发和测试。

[https://www.google.com/chrome/](https://www.google.com/chrome/)

#### Postman

API 接口联调和测试的工具，可管理项目 API、Mock API 实现、团队化管理 API 接口、接口文档化等。

[https://www.getpostman.com](https://www.getpostman.com)

#### 欧陆词典

单词翻译工具

[http://cidian.youdao.com/mac/](http://cidian.youdao.com/mac/)

#### Sequelpro

MySQL 免费数据库连接客户端。

[http://www.sequelpro.com/]](http://www.sequelpro.com/])

#### Android File Transfer

连通PC 连接 Android 手机，将 Android 的文件展示出来。便于文件的维护和管理。

[https://www.android.com/filetransfer/](https://www.android.com/filetransfer/)

#### Chrome Remote Desktop

Chrome 插件，用于远程连接其它电脑，需要被连接电脑中也安装 Chrome 和插件。

[https://chrome.google.com/webstore/detail/chrome-remote-desktop/gbchcmhmhahfdphkhkmpfmihenigjmpp](https://chrome.google.com/webstore/detail/chrome-remote-desktop/gbchcmhmhahfdphkhkmpfmihenigjmpp)

#### Cisco Anyconnect

Cisco 的 VPN 连接工具。

[https://www.cisco.com/c/en/us/products/security/anyconnect-secure-mobility-client/index.html](https://www.cisco.com/c/en/us/products/security/anyconnect-secure-mobility-client/index.html)

#### Docker

容器Docker 的工具，用于管理、维护、发布 Docker 文件。

[https://www.docker.com/](https://www.docker.com/)

#### Genymotion

管理和维护Android 的模拟器。

[https://www.genymotion.com/](https://www.genymotion.com/)

#### Google Drive File Stream

将 Google Drive 中的文件通过文件通过文件资源管理器的方式展现。

[https://www.google.com/drive/download/](https://www.google.com/drive/download/)

#### JProfile

Java 的性能监测工具

[https://www.ej-technologies.com/products/jprofiler/overview.html](https://www.ej-technologies.com/products/jprofiler/overview.html)

#### Parallels Desktop

虚拟机管理工具，支持将虚拟机中的程序与本机程序平滑切换。

[https://www.parallels.com/products/desktop/](https://www.parallels.com/products/desktop/)

#### Robo 3T

Mongo 数据的客户端连接工具

[https://robomongo.org/](https://robomongo.org/)

#### Zoom

免费的视频会议软件，支持视频分享、录制。

[https://zoom.us/](https://zoom.us/)

#### WeChat

微信，聊天和办工沟通。

[http://weixin.com/](http://weixin.com/)
